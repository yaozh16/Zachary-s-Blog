# 修改
```
\033[显示方式;字体色;背景色m......\033[0m
```

每一个
```
\033...m
```
标识一个样式的开始
具体内容如下
| 显示方式 | 显示方式说明 |
|:---------|:-------------|
| 0        | 终端默认设置 |
| 1        | 高亮显示     |
| 2        | 变暗         |
| 3        | 倾斜         |
| 4        | 使用下划线   |
| 5        | 闪烁         |
| 7        | 反白显示     |
| 8        | 不可见       |
| 9        | 删除线       |

| 字体色 | 背景色 | 颜色描述 |
|:-------|:-------|:---------|
| 30     | 40     | 黑色     |
| 31     | 41     | 红色     |
| 32     | 42     | 绿色     |
| 33     | 43     | 黃色     |
| 34     | 44     | 蓝色     |
| 35     | 45     | 紫红色   |
| 36     | 46     | 青蓝色   |
| 37     | 47     | 白色     |

```
"\033[?25l"    隐藏光标
"\033[?25h"    显示光标
```
